# Як запустити TradeDash 🚀

TradeDash — це мультивалютний торговий дашборд із захищеним входом через MetaMask.

### 1. Підготовка
- **Node.js**: Має бути встановлений у вашій системі.
- **MetaMask**: Встановлене розширення у браузері.
- **Майстер-пароль**: При першому запуску вам потрібно буде створити майстер-пароль (або використати MetaMask), щоб захистити дані.

### 2. Запуск бекенд-проксі
Бекенд обробляє запити до бірж (Variational, Nado, Extended).
```powershell
npm install
npm start
```
*Сервер запуститься на порту: 3000*

### 3. Доступ до фронтенду
Відкрийте дашборд у браузері:
- Перейдіть на головну сторінку: **http://localhost:3000**
- Або відразу в дашборд: **http://localhost:3000/dashboard**

### 4. Початкова ініціалізація
1.  **Розблокувати сховище**: Ви побачите запит "Secure Vault". Натисніть "**UNLOCK_WITH_METAMASK**", щоб зайти за допомогою гаманця.
2.  **Додати гаманець**: Натисніть "Link Wallet".
3.  **Авторизація**: Виберіть біржу та натисніть "**🔒 Authorize via MetaMask**".
4.  **Приватний ключ**: Якщо вам потрібні автоматизовані дії, введіть приватний ключ у відповідне поле — він буде **зашифрований локально**.

> [!IMPORTANT]
> **Примітка щодо помилок API**: Якщо при запуску `npm start` ви бачите помилки (403 або 404), це нормально, оскільки деякі дані потребують ваших власних API ключів або підписів, які ви додасте вже у самому дашборді.
